Detailed Description

The TileGenerator class is a Unity Component that can be placed on any Unity object. When given worldY/XSize values and a tile prefab, it will spawn those tile prefabs in an X by Y grid. The prefab tile should also have its own script to help it determine what variant/style it should use.
