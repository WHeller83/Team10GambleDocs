Detailed Description

The DebugMenu script creates a cuztmoizable debug menu that can track variables and be assigned buttons that can call functions at any point in the game. Details are as follows:

Trackers - Variables that you can show on the debug menu that will update every frame. Far easier than the inspector due to the great number of objects that will be in our scene, and the fact that these variables might be among many objects. Delegates - Functions that you can pass to DebugMenu that will make a button, letting you call the public void function from a ui, no matter where the function is in the game. (.NET framework allows this)

To make a Tracker, you need to make a public string function that returns the value you wish to track as a string. (C# is strongly typed. Then, you must call DebugMenu.DM.AddTracker(string name, function); The tracker will then be displayed. To make a Delegate function, you need to make a public void function. It cannot take or return parameters but can interface with your script freely. Then, you must call DebugMenu.DM.AddButton(string name, function); The button will then be displayed.

Removing Trackers and Delegates can be done by the RemoveTracker and RemoveButton functions, keyed by the name used.

Examples is shown in the MoneyTracker.cs script, and displayed on the in-game UI.
